## User Profile Form (tkinter)

This is a simple GUI project I built using Python tkinter while learning how to create desktop applications.

The idea was to make a basic user form where someone can enter their details like :-

(1) name

(2) gender

(3) course

(4) experience level

I also added a basic validation to make sure all required details are filled before submitting.

## What this project does

- Takes user input
- Lets user select gender using radio buttons
- Provides course and experience options using dropdowns
- Includes a terms & conditions using checkbox
- Shows a result after validation
- Has a reset button to clear everything
